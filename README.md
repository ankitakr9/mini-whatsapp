# mini-whatsapp-chat-App
# 💬 Mini WhatsApp Chat Application

A simple WhatsApp-inspired chat application built using Node.js, Express.js, MongoDB, EJS, and HTML/CSS.

---

## 🚀 Features

- 📝 Create new chat messages
- ✏ Edit existing messages
- 🗑 Delete chats
- 📅 Display timestamp for each message
- 👤 Show sender and receiver name
- 📂 RESTful routing implementation

---

## 🛠 Tech Stack

- Node.js
- Express.js
- MongoDB (Mongoose)
- EJS (Template Engine)
- HTML5
- CSS3

## ⚙️ Installation & Setup

1. Clone the repository


git clone https://github.com/your-username/mini-whatsapp-chat.git


2. Install dependencies


npm install


3. Start MongoDB locally OR connect to MongoDB Atlas

4. Run the server


nodemon index.js


5. Open in browser:



---

Make sure to:
- Add MongoDB Atlas connection string
- Set environment variables
